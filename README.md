Android-Weather-Application
===========================

Android based weather application that allows the user to add cities to their main dashboard for live weather updates. Live weather updates are pulled using JSON from Wunderground.com using ISO codes, users can also auto detect their own location using GPS or WIFI. Each segment also allows the user to change the colour of any of their cities, each city is saved to the userâ€™s phone using SQLite. In addition to being able to change the colour of any city segmentation, there is also a PSD file for the full design. Lastly, this application also allows you to post to Twitter using the Twitter API.
